Scene - iOS Application

Scene is a location-oriented iOS application designed for college campuses. it records the location of users and anonymously plots all users to a campus map. Users can identify which locations on campus are busy, thus alleviating congested scenes. This was the first project completed for the Dartmouth Programming Collaborative. Coming soon to the app store.
